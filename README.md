## 📝 소개

1. **어떤 문제를 해결하고 싶은지 정의**
    
        손글씨 이미지에서 텍스트 추출하여 텍스트의 언어를 탐지 후 텍스트를 한국어로 번역해서 출력

    - Text Classification
    - Translation
      
3. **task 별 모델들을 가져와 적합한 모델 찾기**

    - papluca/xlm-roberta-base-language-detection
